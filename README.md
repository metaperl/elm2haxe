# elm2haxe

Implementation of [the Elm Language
examples](http://elm-lang.org/Examples.elm) in Haxe.

## Running Examples

Examples require [Haxe 3.2.0](http://haxe.org/download/). OpenFL examples use OpenFL 3.0.8, Lime 2.4.3.

```
make [exercise name]
```

example:

```
make js.image
```
